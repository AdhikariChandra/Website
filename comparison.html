Haskell vs. Java

Haskell Java
Produces more memory management due to immutable variables. Garbage is collected using the Glasgow Haskell Compiler (GHC). Java Objects reside within the heap, garbage is collected using the JVM when the heap has reached its capacity.
Using the Monad mathematical foundation. All I/O actions return a value tagged with the 'IO' type to keep it I/O values seperate from other values, increasing the purity of the language. Uses I/O streams to determine input or output destination. Streams can support several different kinds of data types.
